##### 2.2

HTTP定义了web客户端向web服务端请求web页面的方式，以及服务器向客户端传送web页面的方式。

HTTP使用TCP作为它的支撑传输协议。HTTP客户端首先发起一个与服务器的TCP连接。一旦连接建立，客户端和服务端进程就可以通过套接字接口访问TCP。

HTTP是一个无状态的协议（stateless protocol）

##### 2.2.3

HTTP报文格式：

- 请求报文

  第一行为**请求行**（request line），包含字段：

  - 方法字段
  - URL字段
  - HTTP版本字段

- 响应报文
